1) Constructing an Input File

 

A GAMESS input file can be constructed in either of two ways:

 

         (1) automatically by selecting items from the interface menu

 

         (2) manually by typing directly into the input file window

 

Each item that is selected from one of the interface menus will be inserted automatically into the input file within the appropriate $GROUP heading.

 

Certain items have not yet been added to the interface menus. If an item that you want to add to the input file cannot be selected from an interface menu, that item can be added simply by manually entering it into the GAMESS input file window.

 

Note: Currently all manual editing of the input file should only be done after all menu item selections have been made to prevent the possibility of some manual entries being overwritten. This is a bug in the interface that will be corrected in the next release of GridChem.

 

 

 

2) Molecular Display

 

The structure of your molecule will be displayed in the Molecular Display panel of the GAMESS interface. Selecting the Data submenu item in the Molecular Specification menu on the main menu bar of the interface will bring up a Molecular Specification window in which you will enter the coordinates of your molecule. The coordinates can be entered either as Cartesian coordinates or in Z-matrix format and can either be entered manually by typing directly into the window’s text area or can be copied from another file.

 

Example:

 Z-Matrix format:

C
O 1 1.20
N 1 1.35 2 124.93
H 3 0.99 1 119.16 2 180.00

Cartesian Coordinates:

C 6 0.00 0.00 1.25
O 8 0.00 1.05 1.45
H 1 0.00 0.98 1.16

After clicking the Done button in this window, the molecular specification will automatically be written to the $DATA group in the input file and the molecule will be displayed in the Molecular Display panel.

The molecule can be displayed as a ball-and-stick, wireframe, or space-filling model. The display format can be selected from the Display Options submenu item in the View menu on the main menu bar of the Molecular Display panel.

 

The molecular display can be manipulated in the display panel as follows:

 

                (1) left  and right arrow keys will move molecule to the left or right, respectively

                (2) click and hold down left mouse button to rotate molecule

                (3) click and hold down right mouse button to zoom in or out on molecule

 

The radius of the atoms in the display can be modified by selecting Modify Atom Radius in the Display Options submenu.

 

Since the bond distance between any two atoms varies from molecule to molecule, some molecules may be displayed either with too many bonds or too few bonds. Additional bonds can be added or incorrectly displayed bonds can be eliminated by selecting Modify Connectivity in the Display Options submenu.

 

Bonds can be displayed in color or without color or can be hidden entirely by selecting the Show Bonds or Hide Bonds submenu items, respectively, in the View menu on the main menu bar of the Molecular Display panel.

 

Note: Information about the format of the input file can be found at http://phoenix.liu.edu/~nmatsuna/gamess/input.doc.html
